# Hantavirus Outbreak Tracker

A live dashboard tracking the April 2026 hantavirus outbreak linked to the cruise ship MV Hondius, produced by **Research Collective**.

## 🌐 View Live

Hosted via GitHub Pages: `https://<your-username>.github.io/hantavirus-dashboard`

## 📊 What it tracks

- Confirmed & suspected case counts
- Fatalities and patient outcomes
- Case timeline (April–present)
- Countries monitoring former passengers
- Key scientific facts (strain, transmission, treatment)

## 🚀 Deploy to GitHub Pages

1. Fork or upload this repo to your GitHub account
2. Go to **Settings → Pages**
3. Under **Source**, select **Deploy from a branch**
4. Set branch to `main`, folder to `/ (root)`
5. Click **Save** — your site will be live in ~60 seconds at `https://<your-username>.github.io/<repo-name>`

## 🛠 Tech

Pure HTML/CSS/JS — no frameworks, no build step, no dependencies. Works offline once loaded.
